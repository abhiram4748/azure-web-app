# Azure Cloud E-Commerce Web Application

## Project Description

This project is a full-stack e-commerce website deployed completely on Microsoft Azure Cloud.

## Technologies Used

* Frontend: React.js
* Backend: Node.js + Express
* Database: Azure SQL Database
* Cloud Hosting: Microsoft Azure

## Azure Services Used

* Azure Static Web Apps (Frontend Hosting)
* Azure App Service (Backend API Hosting)
* Azure SQL Database (Cloud Database)
* GitHub Actions (CI/CD Deployment)

## Architecture

The React frontend communicates with the Node.js backend using REST APIs.
The backend securely connects to Azure SQL Database using environment variables.
The database is never accessed directly from the browser, ensuring security.

## Features

* Product browsing
* Add to cart
* Size selection
* Order placement
* Persistent database storage
